Featured Projects

Kayam Tracking App

2023

Problem

In many schools in Sarawak, there are students who are already ten years old but still can't speak or read. To help them, the government decided to send hundreds of tablets to Sarawak. These tablets have the Kayam App, which helps students who are trailing to catch up to the level they're supposed to be at. However, it was very difficult to track the delivery of the tablets since people used to do it manually.

Process

I developed a Python-based registration tool integrated with Google Sheets API so that I could get real-time data. The system automatically logs, verifies, and monitors the tablets that are distributed.

Impact

This made sure that every part of the process was automated, with no mistakes or missing data, making it much easier and faster to track all the tablets sent to schools.

Python Google Sheets API Data Management

Map Mileage Calculator

2025

Problem

In addition to the Kayam Tracking App, the drivers who delivered the tablets didn't know how much they would earn for each trip. There was also a risk that some drivers might cheat about the number of kilometers they had driven to get more money.

Process

I built a web-based tool that uses mapping data to automatically calculate the distance and total mileage between locations. It then calculates the driver's earnings based on the number of kilometers covered.

Impact

This helps provide proof of the actual distance driven and ensures that each driver gets paid the correct amount based on accurate data.

Web Development Mapping API Data Analysis

Chumbaka Student Project Report Tool

2023

Problem

Chumbaka didn't have a proper system to analyze and showcase the good projects that students had completed. Everything had to be done manually, which took a lot of time and effort.

Process

I created an AppSheet-based system integrated with Google Sheets. Teachers can submit student projects through a digital form that includes photos and descriptions. The tool then automatically compiles and generates personalized reports for each student.

Impact

At the end of each term, a PDF report is automatically generated to show each student's progress and achievements, making it easier for teachers to highlight outstanding projects.

AppSheet Google Sheets Report Generation
